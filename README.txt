Instrucciones rápidas para publicar:
1) Netlify Drop (más simple):
   - Abre https://app.netlify.com/drop
   - Arrastra y suelta el archivo zip (sunflower_surprise.zip) o la carpeta
   - Netlify generará una URL pública inmediatamente.

2) GitHub Pages:
   - Crea un repositorio nuevo en GitHub.
   - Sube index.html al repo (en la rama main).
   - En Settings -> Pages, activa 'Deploy from branch' y selecciona 'main' / / (root).
   - Espera unos minutos y visita: https://TUUSUARIO.github.io/TU-REPO/

3) Glitch (edición en línea):
   - Abre https://glitch.com/
   - Crea un nuevo 'hello-webpage' y reemplaza el contenido de index.html por este.

Archivo incluido:
- index.html (self-contained)
